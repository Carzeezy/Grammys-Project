# Grammys-Project
# Website Analytics Dashboard in Microsoft Excel

## Project Overview

This project analyzes website performance data for Grammy.com and RecordingAcademy.com following their 2022 website split. Using Microsoft Excel, I evaluated user engagement metrics and mobile visitor trends to assess the business impact of separating the two websites.

## Business Question

How did the 2022 split of Grammy.com and RecordingAcademy.com affect website engagement and mobile user traffic?

## Tools & Skills

- Microsoft Excel
- PivotTables
- XLOOKUP
- IF Statements
- SUMIFS
- AVERAGEIFS
- Conditional Formatting
- Data Cleaning
- KPI Analysis
- Dashboard Development

## Analysis

Using advanced Excel functions and analytical techniques, I:

- Calculated key user engagement KPIs
- Measured mobile visitor rates across both websites
- Compared performance metrics before and after the website split
- Identified trends in visitor behavior and engagement
- Built an interactive dashboard to summarize findings

## Key Insights

- Compared engagement metrics between Grammy.com and RecordingAcademy.com after the 2022 split.
- Evaluated changes in mobile visitor traffic.
- Identified performance trends that helped assess the business impact of the website separation.
- Presented findings through a visual Excel dashboard.
